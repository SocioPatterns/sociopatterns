This dataset contains contact networks and demographic information for almost 500 participants in a large-scale SARS-CoV-2 household transmission carried out in South Africa in 2020-2021. The study was led by Jackie Kleynhans at <a href="https://www.nicd.ac.za" target="_blank">NICD</a>. Results from the analysis of this dataset are available atÂ <a href="https://elifesciences.org/articles/84753" target="_blank">J. Kleynhans et al., eLife 12, e84753 (2023)</a>. Details about the main transmission study are available <a href="https://academic.oup.com/cid/article/76/3/e71/6655792" target="_blank">here</a>.

The data can be downloaded at <a href="https://github.com/crdm-nicd/sashts" target="_blank">this GitHub repository</a>.
